# R---statistical-computation
R is a language which has been widely adopted in the fields of bioinformatics, data analysis, and data science.

Content of this demonstration (Core file: '.Rmd')

1.Applied matrix algebra

2.Applied descriptive statistics analysis

3.Create informative graphs and comment on them

4.Do a regression analysis and comment on it

5.Compare distribution functions and comment on them. Use e.g. descriptive statistics, graphs etc and point out the difference between the distribution functions

6.Do a Monte Carlo Simulation (preferably based on empirical data) and comment on the results

7.Do several Monte Carlo Simulations with identical parameters and compare their outcomes (repectively their differences). Tip: Use the set.seed function to make your results reproducible.

8.Write a function to solve a problem in the fields of finance, risk management or asset management.
